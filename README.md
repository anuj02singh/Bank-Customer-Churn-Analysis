# Bank Customer Churn Analysis

## Objective
To analyze why customers are leaving the bank and identify key churn drivers using Python, SQL and Power BI.

## Dataset
- Source: GitHub (Kaggle)
- Records: 10,000 customers
- Features: 14 columns

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- PostgreSQL (pgAdmin 4)
- Power BI (Coming Soon)
- Google Colab

## Key Findings
- Overall Churn Rate: 20.37%
- Germany highest churn: 32.44%
- Female churn higher: 25.07%
- Inactive members 2x more likely to churn
- High balance customers also churning
